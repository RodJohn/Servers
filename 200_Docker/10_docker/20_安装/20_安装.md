# 6 centos

##  版本说明

    Docker CE 支持 64 位版本 CentOS 7，并且要求内核版本不低于 3.10。
    CentOS 7 满足最低内核的要求，
    但由于内核版本比较低，部分功能无法使用，并且部分功能可能不太稳定。

## 清除历史版本

    yum remove docker \
                      docker-client \
                      docker-client-latest \
                      docker-common \
                      docker-latest \
                      docker-latest-logrotate \
                      docker-logrotate \
                      docker-selinux \
                      docker-engine-selinux \
                      docker-engine -y
